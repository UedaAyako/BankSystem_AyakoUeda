# 💰 BankSystem

Java で構築したシンプルな銀行システムです。  
ユーザーはIDとパスワードでログインし、入出金操作を通じて残高の確認を行うことができます。  
例外処理により、誤操作時にも安全に処理が行われるよう設計されています。

## 🔧 実装機能

- ✅ ログイン機能（ID・パスワード認証）
- 💵 入金機能
- 💸 出金機能
- 🧾 残高の確認
- ⚠️ 不正操作時の例外処理（例：パスワード不一致、残高不足、無効な金額入力など）
